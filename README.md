## ambari-apache-zookeeper-service

Ambari 集成 Apache Zookeeper

分支介绍：
- main: 主分支，不用参考
- hdp_3.0_ZOOKEEPER: 该分支代码为 hdp3.0 版 ZOOKEEPER 集成代码（未做任何修改）。
